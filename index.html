s3cam.js

Description

An HTML5/JS webcam app.

Purely Client-Side
This app does not rely on any server-side code (besides amazon of course).
Motion-Detection
The motion detection works by taking a diff of the current frame and the previous frame.
Straight to Amazon
The images are uploaded to Amazon S3 on-the-fly. Local storage is vulnerable to an attacker taking the device.
